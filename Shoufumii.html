Project

Shoufumii



The Problem

"How can I offer Discord servers more functionality the users can enjoy that also helps them?"

The Solution

Shoufumii is a bot written in C# to grant users in Discord servers more functionalities than the users would have without Shoufumii.
One of the many features Shoufumii provides is actually quite simple. Using the command "?notes" and writing down some text, the user is capable of adding text to a saved text file. At first I developed this feature because it was useful to me to take quick notes of things and easily read all of it back and share it with others in my server.

After testing and using this feature for a while, I decided to have the bot create different files for each user to add their notes to. This was useful so that the other users wouldn't add to your personal notes, which could become quite messy. This feature proved to be more useful than writing notes straight into the chat without using the bot, because over time it'd be much harder to find back the notes one had taken.

Using the "?notes" command one can display their entire list of notations to read back all at once, so the users would never have a difficult time finding back their notes.

Other than that, the bot also includes fun commands to interact with others in the server such as "?attack [@username]" to display a GIF of an attack while mentioning the user that used the command and the user that was mentioned by the user that used the command.

Example:
"?attack @IamNoNamer"

Output:
"Raritoken attacks @IamNoNamer"



The bot has many commands included and continues to be expanded upon, and because it can be extremely overwhelming I created a help command as well so you can view all possible commands per catergory.

There are five major catergories that will be sent through direct/private messages to the user, these catergories are: Command List, Utility Commands, Blade & Soul Commands, Fun Commands and last but not least Moderation Commands.

What have I learned?

During this project I learned slowly but steadily to create more useful and fun commands for users to use in Discord servers.

I also began to start understanding C# better than I previously had, and adding to that, I also learned how to utilize json files to setup the bot in order to make other Discord users capable of hosting the bot themselves and setting up requirements and exceptions to use the bot in certain channels on userID basis.

I've also figured out that I really enjoy solving problems and creating things myself while I've been working on my bot on and off for a few years now, sometimes I even get requests/ideas from friends/others to implement into my bot.
Then it's up to me to figure out how to realize the requested feature in my bot.

A lot of the time I will have to search online to figure out a thing or two to create the features I am working on, so I learnt how to look online for possible solutions for when I struggle with some things in my code or to simply improve my code.
